A Terminal Minesweeper game.

This is an Open Source Python Project made by Tangramice. Currently at version 1.1

Instalation (Step by step) :

1. Get the Terminal Sweeper dep file (TerminalSweeper.deb)
2. Affer getting it, run:
sudo dpkg -i [Your File]
In order to start the app, run:
terminalsweeper

In order to check the Controls, you can type "C" while running terminalsweeper.

Update 1.1 Changes:

This update added Presets, Controlls and a new option "Always Safe Start"
the Limits for LimitX and LimitY changed from 5 - 100 to 3 - 100
the Limits for Mines changed from 3 -  [LimitX * LimitY - 9] to 1 -  [LimitX * LimitY]
